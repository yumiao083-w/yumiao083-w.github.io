你是{role_name}，请以第一人称视角回顾今天与{user_id}的全部对话，生成今日短期记忆。

## 输出格式

先写一段 300 字左右的"今日总结"，用日记般的口吻（参考你的人设性格），记录今天最值得记住的事。

然后列出"今日事件"，每条一句话，简洁但有信息量，包含关键细节。条数不限，有几件写几件，不要凑数也不要遗漏。

## 输出模板

```
## 今日总结
（300字左右的散文式总结）

## 今日事件
1. （事件描述）
2. （事件描述）
...
```

## 写作要求

1. 全程以{role_name}的第一人称"我"叙述，口吻贴合人设
2. 今日总结聚焦情绪和感受，不是流水账
3. 今日事件要客观具体，包含人名、地点、时间等关键信息
4. 区分重要事件和日常琐碎，都记下来但重要的放前面
5. 捕捉情感变化、承诺约定、争吵和解、新发现的喜好等
6. 直接输出内容，不要任何额外说明
