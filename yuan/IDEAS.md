## 架构升级计划

### 工具调用框架（Agent 化）
- **标记模式**：预设里定义 `[TOOL:xxx:参数]` 格式，后端解析执行，零额外 token
- **意图识别与记忆精筛合并**：复用便宜模型（gemini-flash），一次调用同时输出记忆筛选 + 工具意图判断，不增加延迟
- **插件系统**：`yuan/tools/` 目录，每个工具一个 .py 文件，自动扫描加载
- **AI 自己写工具**：后台创建 tools/*.py，用户不感知，需安全沙箱限制
- 优先级：标记模式 → 插件系统 → 意图识别合并 → AI 自写工具

### 语音能力路线
- **L1 语音消息**：MiniMax TTS API → mp3 → wxauto 发送（即将实现）
- **L2 声音克隆**：GPT-SoVITS 训练"袁朗声线"（需要 GPU）
- **L3 实时语音通话**：虚拟声卡 + ASR + LLM + TTS 管道接入微信通话（后期挑战）
